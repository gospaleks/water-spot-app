# 💧 Water Spot

**Water Spot** is an Android mobile application developed in **Kotlin** using **Jetpack Compose**.

The main goal of the app is to enable users to easily find and report free water sources worldwide, with an addition of **gamification** and **community contribution**.  

---

## 🚀 Main Features

### 🗺️ Map and List of Water Sources
- Add new water sources on the map:
  - Choose the type of source (public tap, refill station, water fountain, etc.)
  - Define the cleanliness level
  - Add a description
- View water sources on the map as **Markers**
- View all water sources also in a **list view**
  - Supports **filters, search, sorting, and date range**

### 📍 Interaction with Water Sources
- Add new photos
- Leave ratings and comments
- Mark water sources as **visited** (personal tracking)

### 🎮 Gamification
- Every interaction gives a certain amount of **points**
- Public leaderboard based on points
- Badge system (5 levels):
  - Explorer
  - Contributor
  - Scout
  - Guide
  - Water Guardian

### 🔔 Notifications and Service
- Users can enable a **service**
  - Configure the **search radius**  
  - The service checks if there are new sources or other users nearby  
- **System notifications** (delivered even when the app is not active)

### 🧭 Filters and Search
- **On the map:**
  - Source type
  - Cleanliness level
  - Search radius
- **In the list:**
  - All map filters
  - Search (by any text)
  - Date range
  - Sorting

---

## 🛠️ Technologies and Architecture

- **Android** – Kotlin + Jetpack Compose
- **Clean Architecture + MVVM** – clear separation of layers and responsibilities
- **Firebase**
  - Firebase Auth – user authentication
  - Firebase Firestore – database
- **Cloudinary** – image storage
